Welcome to the Docs @ ja13.org

The Docs are designed to be a reference for myself to be able to replicate what I have created that runs my website, and my server in general, so that I can use it as and example for others, or for when I am setting up other sites. It also reminds me about where everything is, since documentation is important. Obviously, nothing extremely sensitive will be stored here, such as passwords or SSH keys or PGP keys or SSL keys, but besides that, pretty much everything is here in a basic format.

IP Allocations

Static

From 192.168.0.1 through 192.168.0.99 is static IP sets.

  • 10 through 19 is hardware
  • 20 through 79 is VMs
  • 1 through 9 and 80 through 89 is network hardware
  • 90 through 99 is for lan parties
