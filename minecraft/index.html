Map Information

The current map is a mix of some new 1.10 terrain plus on the east side of the map, we have stiched in the older 1.7.4 map. By default, you spawn in at the new spawnpoint in the newer part of the map, but by running the 

/warp oldspawn

command, you can go back to the older map spawn. There is also a large Tardis system, where small blue police boxes show the way. Many of the links are a bit broken however, so beware.

Transportation

Transportation is acieved through many footpaths, rail systems, warps, and the Tardis (which often uses the warps). There is one massive footpath around the entire 1.7.4 map, many kilometers of rail lines that link the old and new maps together, and two tardis's which are linked togeter. The rail systems are best navigated by getting on the track at a station on the right hand side, and the pushing forward. There are a few techniques to navigating some of the stations, but the signage is rather decent and consistent, feel free to ask how to best navigate the system. There are also "european fixers" in many places, which put you on the right side of the track if you accidentally launch from the left.

Server Map History

We have a handful of older maps. There have been over 20 different worlds hosted on the server over 5 different years, but three of the maps have been the longest lived.

1.7.4 Map (7z)

This map lasted for a very long time, starting in September 2014, and running all the way to early 2016. It was the first map to be used for a very long time on the server, maps before this only being up for about a month to a week at a time. There are many landmarks and the history of the map is rich, especially the map spawn. There were 5 primary maps when the cord was cut:

  • main - This map was the spawnpoint, so that if people happened to find the server, they would be away from all of our stuff and not able to grief the main stuff
  • world - This map was the primary map where everything was made. There were warps to the main, mesa, and other maps all in the spawn area.
  • mesa - This map was a late map created in likely late 2015 when my server went down for two weeks due to a broken GRUB, and we used this map in place of the other map.
  • world-nether - This map was a custom parkour map mostly unmodified from when I downloaded it.
  • world-the-end - This map mostly contains Eonaut's big end fortress.

1.9.4 Map (7z)

This map was not running for very long (likely from early 2016 to June 16, 2016), but has a handful of builds by jrddunbr and Eonaut. jhndunbr also built mostly in the "Creative area" on the other side of the "creative wall" which was partially a wall, and mostly a row of beacon buildings.

1.10 Map (zip)

This map was run from June 16, 2016 to early July 2016, this world was created when the server first went public.

Current Map

This map was started in mid 2016, and has been running since. It's a stitched version of the 1.7.4 main world, the Design warp (from 1.7.4's Mesa world), and a brand new 1.10 4k by 4k map, with a few major biomes: Mountain range, field, snow mountains, savanah, rolling hills, and desert mesas.
