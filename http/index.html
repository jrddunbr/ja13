Services

There are a few services provided by this site. These services are for the most part on virutal machines for isolation. These virtual machines run Arch Linux, with two exceptions being the two "MediaWikis".

Services include the web server (not to mention a reverse proxy as well), an instance of "management2" (check out GitHub), two Minecraft servers, my resume and my pgp keys (so that people may gather them as they please), and finally, two wikis, one for the public Minecraft server and the other which documents how I installed all of the virtual machines, how they are configured, and how to recreate them or to make new virtual machines.

I highly recommend going to the docs and reading about how I set this website up, particularly if you are a person who likes technical computer stuff. (not to mention that I set it all up in the proximity of two days)
